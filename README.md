# SOC Playbooks

Praktische SOC-Playbooks fuer wiederholbare technische Analysen und Incident-Workflows.

## Inhalt
- `playbooks/wireshark.md`
- `playbooks/ssl-tls-check.md`
- `playbooks/nmap.md`
- `playbooks/hash-cracking.md`

## Ziel
Dieses Repository liefert standardisierte Schritt-fuer-Schritt-Prozesse fuer:
- Netzwerk-Paketanalyse (Wireshark)
- TLS/SSL-Pruefungen
- Host- und Service-Enumeration (Nmap)
- Hash-Analyse und Passwortwiederherstellung im forensischen Kontext

## Wichtige Hinweise
- Nur auf Systemen mit expliziter Berechtigung einsetzen.
- Ergebnisse immer in Tickets/Case-Management dokumentieren.
- Bei kritischen Findings sofortige Eskalation nach internem IR-Prozess.
